# Theme Packs

This directory contains reusable presentation-only theme packs.

Included starter themes:

- `default`
  - neutral light theme
- `corporate-blue`
  - branded enterprise theme
- `dark`
  - high-contrast dark theme

Each theme pack includes:

- `theme.yaml`
- `tokens.yaml`
- `assets/`
- `README.md`

Themes affect visual presentation only. Course structure remains in templates, variants, and course projects.
