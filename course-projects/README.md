# Course Projects

This directory contains starter Course-as-Code projects that can be opened directly in the studio or built from the CLI and GitHub Actions workflows.

Included starter projects:

- `security-awareness`
  - multi-variant phishing-awareness family
- `workplace-conduct`
  - multi-variant conduct and reporting family
- `customer-service`
  - customer-service escalation starter scenario

Each project keeps:

- `project.yaml`
- template source
- variant YAML files
- theme packs
- project assets
- `build/README.md`

Generated SCORM packages belong under each project `build/` directory and should remain build artifacts.
