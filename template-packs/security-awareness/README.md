# Security Awareness Pack

Use this pack to keep one phishing or refresher template in Git and rebuild multiple organization-specific variants from different variable sets.

Recommended workflow:

- choose a template
- choose or duplicate a variable set
- preview the compiled variant
- export a SCORM build

Source stays in control:

- shared template YAML defines the branching structure
- variant YAML files carry organization-specific values
- generated SCORM packages remain build artifacts
