# Template Packs

This directory contains reusable template packs for repeatable course families.

Included starter packs:

- `security-awareness`
- `workplace-conduct`
- `customer-service-escalation`

Each pack includes:

- `pack.yaml`
- `README.md`
- one or more variant YAML files
- references to shared template source and schema files

Use template packs when the same branching structure should produce many variants from different variable sets.
